## 🚀 Getting Started Locally

### Prerequisites
- XAMPP / WAMP installed
- PHP 7.4+
- MySQL 5.7+

### Installation

```bash
# Step 1 — Clone the repository
git clone https://github.com/richapandey95/travel-canvas-holiday.git

# Step 2 — Move to XAMPP htdocs folder
mv travel-canvas-holiday /xampp/htdocs/

# Step 3 — Import Database
# Open phpMyAdmin → Create DB "travel_canvas" → Import travel_canvas.sql

# Step 4 — Configure Database
# Edit includes/db.php
$host = "localhost";
$user = "root";
$pass = "";
$db   = "travel_canvas";

# Step 5 — Run
# Open browser → http://localhost/travel-canvas-holiday
```
